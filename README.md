# Data Pipeline Project

This project extracts cryptocurrency price data from an API, transforms the data using Python and pandas, stores it in a SQL database, and exports processed data to CSV reports.

## Tech Stack
- Python
- Pandas
- Requests
- SQLite

## Project Structure
- extract.py → fetch data from API
- transform.py → clean and process data
- load.py → save data to SQL and CSV
- main.py → run the full pipeline-pipeline-project
